Installation
=======================

Dependencies
-----------------------
* `matplotlib >=3.3.1`
* `pandas`
* `numpy`
* `scipy`
* `loguru`
* `anndata`
* `fire`
* `scanpy`
* `PyComplexHeatmap`
* `seaborn`
* `adjustText`
* `plotly`
* `ipywidgets`

How to install?
----------------------
1. **Install using pip**  ::

    pip install adataviz
    #or
    pip install --upgrade adataviz

2. **Install the developmental version directly from [github](https://github.com/DingWB/adataviz)**  ::
[https://github.com/DingWB/adataviz](https://github.com/DingWB/adataviz)
pip install git+https://github.com/DingWB/adataviz

if you have installed it previously and want to update it, please run  ::

    pip uninstall adataviz

and install from github again

OR ::

    git clone https://github.com/DingWB/adataviz
    cd adataviz
    python setup.py install
